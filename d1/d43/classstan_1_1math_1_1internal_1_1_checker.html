stan::math::internal::Checker< F, E > Class Template Reference

Detailed Description

template<typename F, typename E>
class stan::math::internal::Checker< F, E >

Apply an error check to a container, signal failure by throwing.

Apply a predicate like is_positive to the double underlying every scalar in a container, throw an exception if the predicate fails for any double.

Template Parameters
Ftype of predicate
Etype of exception thrown
