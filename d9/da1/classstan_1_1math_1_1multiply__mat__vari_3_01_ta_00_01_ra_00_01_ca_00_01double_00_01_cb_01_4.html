Detailed Description

template<typename Ta, int Ra, int Ca, int Cb>
class stan::math::multiply_mat_vari< Ta, Ra, Ca, double, Cb >

This is a subclass of the vari class for matrix multiplication A * B where A is N by M and B is an M by K matrix of doubles.

The class stores the structure of each matrix, the double values of A and B, and pointers to the varis for A and B if A or B is a var. It also instantiates and stores pointers to varis for all elements of A * B.

Template Parameters
Tatype of elements in matrix A
Ranumber of rows in matrix A, can be Eigen::Dynamic
Canumber of columns in matrix A and rows in matrix B
Cbnumber of columns in matrix B, can be Eigen::Dynamic
