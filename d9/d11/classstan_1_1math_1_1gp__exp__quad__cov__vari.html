Detailed Description

template<typename T_x, typename T_sigma, typename T_l>
class stan::math::gp_exp_quad_cov_vari< T_x, T_sigma, T_l >

This is a subclass of the vari class for precomputed gradients of gp_exp_quad_cov.

The class stores the double values for the distance matrix, pointers to the varis for the covariance matrix, along with a pointer to the vari for sigma, and the vari for length_scale.

T_sigmatype of sigma
T_ltype of length scale
